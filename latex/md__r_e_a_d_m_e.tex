El proyecto contiene los códigos del servidor y el cliente. Para ejecutar el proyecto siga los siguientes pasos\+:


\begin{DoxyItemize}
\item Paso 1\+: Ejecute el siguiente comando para compilar todo y mejorar las clases de base de datos\+: \begin{DoxyVerb}mvn clean compile
\end{DoxyVerb}

\item Paso 2\+: Asegurese de que la base de datos se haya configurado correctamente. Abra el script {\itshape HLHotelesbd.\+sql} en My\+SQL Work\+Bench y ejecutelo, para crear la base de datos y otorgar privilegios.
\item Paso 3\+: Ejecute el siguiente comando para crear un esquema de base de datos para esta muestra. \begin{DoxyVerb}mvn datanucleus:schema-create
\end{DoxyVerb}

\item Paso 4\+: Carge los datos de muestra en la base de datos. Abra el script {\itshape ejemplos-\/hoteles.\+sql} en My\+SQL Work\+Bench y ejecutelo.
\item Paso 5\+: Para iniciar el servidor, ejecute el siguiente comando\+: \begin{DoxyVerb}mvn jetty:run
\end{DoxyVerb}

\item Paso 6\+: Ahora, el codigo del cliente se puede ejecutar en una nueva ventana de comandos con \begin{DoxyVerb}mvn exec:java -Pclient
\end{DoxyVerb}
 
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{Pruebas unitarias}\label{md__r_e_a_d_m_e_autotoc_md1}
